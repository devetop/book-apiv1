# Submission Bookshelf API

Deskripsi:
Menyiapkan 1 Instance menggunakan AWS Lightsail. Install NodeJS v18 lalu install hapi.
Buat API dengan beberapa kriteria seperti,
1. Untuk menyimpan buku
2. Menampilkan seluruh buku
3. Menampilkan detail buku
4. Mengubah data buku
5. Menghapus buku

OS yang digunakan:
* Ubuntu 20.04 LTS

Detail Spesifikasi:
* 2 GiB Memory
* 1 vCPU
* 60 GiB SSD

Instance terpasang IP Private 172.26.14.19 dan IP Public 52.221.231.128

Instalasi:

Installing Node.js
```sh
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash - &&\
sudo apt-get install -y nodejs
```

Installing Build Tools
```sh
sudo apt-get install -y build-essential
```

Install hapi dengan npm
```sh
npm install @hapi/hapi
```

Buat file server.js di folder src
```sh
mkdir src
nano src/server.js
```
Lalu edit file server.js
```plaintext
const Hapi = require('@hapi/hapi');
 
const init = async () => {
    const server = Hapi.server({
        port: 5000,
        host: 'localhost',
    });
 
    await server.start();
    console.log(`Server berjalan pada ${server.info.uri}`);
}
 
init()
```

Edit file package.json menjadi seperti berikut
```plaintext
{
  "dependencies": {
    "@hapi/hapi": "^21.3.0"
  },
  "name": "submission",
  "description": "",
  "version": "1.0.0",
  "scripts": {
    "start": "node src/server.js"
  },
  "keywords": [],
  "author": "",
  "license": "ISC"
}
```

Test web server
```sh
npm run start
```
